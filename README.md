# Movable Type Writer
Yet Another Entry Post Tool feat. Movable Type Data API with Google Chrome

Feel free to fork this project!

## Install & build

#### 1. install gulp and related modules
```sh
$ npm install
```

#### 2. Install bower modules
```sh
$ bower install
```

#### 3. Build package
```sh
$ gulp
```
